# eda-leda

[![Java Version](https://img.shields.io/badge/Java-11%2B-magenta?style=flat-square&logo=java)](https://www.oracle.com/java/)
[![Course](https://img.shields.io/badge/Disciplina-LEDA-blue?style=flat-square)](https://www.ufcg.edu.br/)

Este repositório reúne as implementações e resoluções de problemas desenvolvidas ao longo da disciplina de **Laboratório de Estrutura de Dados e Algoritmos (LEDA)** da Universidade Federal de Campina Grande (UFCG).

O objetivo central é o desenvolvimento de algoritmos eficientes e a construção manual de estruturas de dados essenciais, consolidando o entendimento do gerenciamento de memória, complexidade assintótica e raciocínio lógico-abstrato.


## 🔗 Organização

O repositório está estruturado em três grandes pilares de desenvolvimento:

* **Implementações:** Desenvolvimento de algoritmos fundamentais de ordenação, servindo como base sólida de código reutilizável e para relembrar.

* **Questões do `tst-leda`:** Resolução de problemas práticos divididos por tópicos conceituais, focando na aplicação direta de estruturas lineares, não-lineares e ordenação espacial/temporal.

* **Roteiros Práticos:** Projetos estruturados a partir de templates base fornecidos pela disciplina para o desenvolvimento guiado.


### 📂 Questões

| Módulo | Descrição Técnica |
| :--- | :--- |
| `introducao` | Exercícios de nivelamento, manipulação de arrays e recursão básica. |
| `ordenacao-quadratica` | Algoritmos de ordenação com complexidade $O(n^2)$ *(Bubble Sort, Selection Sort, Insertion Sort)*. |
| `ordenacao-logaritmica` | Algoritmos de divisão e conquista com complexidade ${O}(n log n)$ *(Merge Sort, Quick Sort)*. |
| `ordenacao-linear` | Algoritmos de ordenação baseados em chaves com complexidade ${O}(n)$ *(Counting Sort, Radix Sort)*. |
| `estruturas-lineares` | Implementações de Pilhas, Filas e Listas Encadeadas. |
| `arvores` | Resolução de problemas e conceitos sobre Árvores Binárias de Busca (BST) e Heaps. || `arvores` | Resolução de problemas e conceitos sobre Árvores Binárias de Busca (BST) e Heaps. |
