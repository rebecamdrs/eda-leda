# 🌺 tst-leda

Este repositório reúne implementações em Java desenvolvidas ao longo da disciplina de Estruturas de Dados e Algoritmos (LEDA) da UFCG. Os problemas abordam conceitos fundamentais de programação, com ênfase em manipulação de arrays, ordenação e construção de algoritmos eficientes.

As soluções são implementadas manualmente, sem uso de métodos prontos de ordenação ou estruturas auxiliares, priorizando o entendimento dos processos internos dos algoritmos e o desenvolvimento do raciocínio lógico.

**📎** https://www.notion.so/eda-leda-356cd6f691e180e0b216d0ac9c84d6fb?source=copy_link

## 📂 Lista de questões

| Pasta | Conteúdo |
|---------|---------|
| `introducao` | Questões e exercícios introdutórios |
| `estruturas-lineares` | Pilhas, filas, listas encadeadas e listas duplamente encadeadas |
| `ordenacao-quadratica` | Algoritmos de ordenação O(n²), como Bubble Sort, Selection Sort e Insertion Sort |
| `ordenacao-logaritmica` | Algoritmos de ordenação O(n log n), como Merge Sort e Quick Sort |
| `ordenacao-linear` | Algoritmos de ordenação linear, como Counting Sort e Radix Sort |
